# MultiPoint Gradient - Store Listing

## Extension Name

MultiPoint Gradient - Color Picker & CSS Generator

## Summary

Pick colors from any webpage, sample gradients with a line tool, and export as CSS, Tailwind, SVG, or JSON.

## Description

MultiPoint Gradient is a powerful color picker and gradient generator built for web designers and developers. Extract colors directly from any webpage, build custom gradients, and export production-ready code in seconds.

**Two Picking Modes**

- Point picker: Click anywhere on a page to sample individual colors, one by one
- Linear sampler: Draw a line across any area and automatically capture evenly-spaced colors along the path — perfect for extracting existing gradients

**Four Output Formats**

- CSS: Standard `linear-gradient()` syntax, ready to paste into your stylesheet
- Tailwind CSS: Utility class format with `bg-[linear-gradient(...)]`
- SVG: Full `<linearGradient>` XML definition for vector graphics
- Raw JSON: Structured color data for custom tooling and scripts

**Full Gradient Control**

- Visual angle picker with a dial and numeric input (0-359 degrees)
- Reorder, add, or remove color stops at any time
- Live gradient preview updates as you make changes
- View colors in both hex and HSL formats

**Save and Reuse**

- Save gradients to your local history with one click
- Reload any previously saved gradient instantly
- History stores up to 100 entries with visual previews and timestamps

**Designed for Your Workflow**

- Light, dark, and system theme options
- Pixel-accurate sampling that handles high-DPI displays
- Lightweight popup that stays out of your way
- No accounts, no sign-ups, no data collection

## Privacy

This extension does not collect, transmit, or store any user data outside of the browser. All data (color history, settings) is stored locally using the browser's built-in storage API.
